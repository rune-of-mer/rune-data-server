# rune-data-server

Rune of Mer のゲームデータを提供する API サーバ 

```shell
git clone git@github.com:rune-of-mer/rune-data-server.git
cd rune-data-server

cargo build
```

```shell
# Latest Release
docker pull ghcr.io/rune-of-mer/rune-data-server:latest

# Minor Release
docker pull ghcr.io/rune-of-mer/rune-data-server:v0

# Specific Release
docker pull ghcr.io/rune-of-mer/rune-data-server:v0.20.0
```
